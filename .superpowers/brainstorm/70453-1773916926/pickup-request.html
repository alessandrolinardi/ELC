Key improvements:

  • Card-based grouping — Carrier+Date, Address, Packages are visually distinct sections instead of one endless scroll
  • Carrier as visual cards — big clickable tiles instead of tiny radio buttons
  • Address as a summary card — shows selected address inline with a "Change" dropdown, no separate expander for address book
  • Dimensions on one line — "40 × 30 × 20" instead of 3 separate number inputs labeled L/W/H
  • Pallet as a toggle — clean switch instead of a checkbox + conditional section
  • Sticky summary bar — summary + submit always visible together at the bottom, no separate sections
  • Notes field removed from default view — available via "+ Aggiungi note" link to reduce form length
