How it works:

  • Single visual container — looks like one field, but each dimension is an independent number input
  • Labels above each value — Lunghezza / Larghezza / Altezza, so no ambiguity
  • × separators — read naturally as "40 × 30 × 20 cm"
  • Unit label at the end — "cm" shown once, not repeated
  • Per-field validation — if one dimension is missing, only that sub-field highlights red
  • Tab between fields — Tab key moves L → W → H naturally
