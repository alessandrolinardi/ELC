A — Dark Sidebar + Step Wizard

Persistent left sidebar with categorized tools. Each tool uses a step-by-step wizard (① ② ③ ④) so users always know where they are. Dark theme feels modern and reduces visual noise. Scales well — new tools just slot into sidebar categories.

Pros

  • Clear navigation that scales to 10+ tools
  • Step wizard eliminates "what do I do next?" confusion
  • Modern, distinctive look

Cons

  • Dark theme may feel unusual for internal tools
  • Sidebar takes horizontal space

B — Light Top-Nav + Warm Accents

Horizontal top navigation with tab-style tool switching. Light, airy design with warm gold accents (subtle ELC nod). Centered content with clear progress breadcrumbs. Feels like a polished SaaS product. Tools overflow into a "More" dropdown as they grow.

Pros

  • Familiar SaaS feel — zero learning curve
  • Maximum content width (no sidebar eating space)
  • Warm gold accents echo ELC brand subtly

Cons

  • Top nav gets crowded past 6-7 tools
  • Less room for tool categories/grouping

C — Dashboard Home + Drill-In

Home screen shows all tools as a card grid (like a launchpad). Click a tool to "drill in" to its dedicated page with a back button. Scales infinitely — just add more cards. Includes usage stats and recent activity on the home screen.

Pros

  • Scales to any number of tools effortlessly
  • Home dashboard gives overview / quick stats
  • Most natural for a growing "toolbox" concept

Cons

  • Extra click to reach a tool (can't jump directly)
  • No persistent navigation while inside a tool

💡 We can also mix elements — e.g., Dashboard home (C) with top-nav once inside a tool (B). Click your favorite direction to start.
